How it works:
1. Input the binary operators, the power of the base-2 exponent, and the desired number of bits in the output.

2. Decide on the rounding choice. GRS uses Guard, Round, and Sticky bits, which preserves a fixed number of extra bits, while Rounding uses IEEE-754 Rounding rules.

3. Click the add binary button. The page will display the aligned binary operators, the bit display and the answer. If desired, clicking the download result button will download the result in "output.txt"
